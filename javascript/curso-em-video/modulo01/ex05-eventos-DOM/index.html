Eventos DOM

Eventos:

Tudo aquilo que possa acontecer com o meu elemento, no caso a tag DIV.

O que pode acontecer com essa DIV? O mais comum são eventos de mouse

Analisando o código.... Quando eu clicar na minha div será criado um objeto chamado 'a' e ele vai ser uma ligação do objeto 'a' do JavaScript com o objeto de id 'area' que está dentro do HTML

        
            div id="area" onclick="clicar()">
           interaja...
           /div
interaja... 
        
            script
                function clicar() {
                    var a = window.document.getElementById('area')
                    a.innerText = 'Clicou!'
                }
            /script

Alguns tipos de eventos que podem acontecer com o mouse na DIV

  • mouseenter: Momento em que o mouse entra na div
  • mousemove: Dispara sempre que o mouse é movido dentro da div
  • mousedown: Quando aperta e segura o botão do mouse
  • mouseup: Quando está segurando o botão e solta
  • click: Quando o movimento é de click único
  • mouseout: Quando o mouse sai da div
