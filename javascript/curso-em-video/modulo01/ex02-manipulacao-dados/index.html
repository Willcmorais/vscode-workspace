Código exemplo para prompt

A variável 'numero' recebe o resultado do prompt do que aparecer na janela

Uma característica do comando window.prompt é que ele retorna uma string, então, sempre que utilizamos o + para somar ele vai concatenar. Para somar utilizamos uma conversão de tipo.

  • var nome = window.prompt("Informe seu nome:");
  • Number.parseInt(n). Hoje utilizamos apenas Number(n)
    • Antes de jogar na variável transforma o que for digitado em um tipo number
  • Para converter algo para string usamos o String(n) ou n.toString

Mostra o que foi digitado

  • window.alert("O nome digitado foi " + nome);
  • O + tem sentido de concatenar (number + number) ou (string + string)
