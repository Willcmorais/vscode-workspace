HTML e CSS box model

Aplicável ao HTML e CSS. Em HTML tudo que é exibido é em forma de caixas, e essas caixas podem ter caixas dentro delas, formando assim uma aninhamento, um tipo de hierarquia entre elas.

  • Border é a linha que circunda o conteúdo.
  • Padding é o preenchimento do elemento, para que a borda fique distante do conteúdo.
  • Margin(fora do elemento) é um espaço externo da linha que circunda o conteúdo.
  • Outline(dentro do elemento) é o contorno dentro da margem.

Tipos de caixas

box-level e inline-level

  • O box level sempre se inicia em uma linha nova e sempre ocupa 100% da viewport
