Tamanho de fontes e suas medidas

Existem várias formas de colocar um font-size, temos medidas

  • Absolutas:
    • cm, mm, in(polegada), px, pt, pc
  • Relativas:
    • em
      • medida relativa a altura "M" da fonte
    • ex
      • medida relativa ao altura "x" da fonte
    • ren, vw (view width, largura da tela), vh (altura da tela), %

As medidas podem ser representadas em vários tipos de telas diferentes, então, o mais recomendado para tamanhos de fontes é o "px e em"

O tamanho padrão das letras é de 16px, geralmente, 1em é igual a 16px. O em é mais indicado pois trabalhamos com porcentagem do tamanho da fonte padrão, é útil para diferentes tipos de tela (celular, tv, computador)

Peso, estilo e shorthand font

weight - peso, width - largura e height - altura

Os valores aceitos para o weight são

  • lighter
  • normal
  • bold
  • bolder

Podendo ser utilizado também variações numéricas de 100 em 100 até 900. Algumas letras não tem todas as possibilidaes de configuração de peso, a melhor para trabalhar com isso é a work sans.

font-family: 'Work Sans';

font-weight: bolder;

font-size: 3em;

font-style: italic;

pode se resumir a uma shorthand "font:..."

font-style -> font-weight -> font-size -> font-family

Exemplo: 
h1 {
        font: italic bolder 3em 'Work Sans', sans-serif;
    }
